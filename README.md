# PyMineSweeper
This application is a mine-sweeper game which works with PyGame library.
このアプリはマインスイーパーゲームです。

## Controls (操作)
- Arrow keys: Move view
- Left click: Open the unflagged cell
- Right click: Flag/deflag the cell
